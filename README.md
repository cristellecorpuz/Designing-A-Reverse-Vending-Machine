# Designing-A-Reverse-Vending-Machine
This project presents a Reverse Vending Machine (RVM) powered by a custom Arduino-based controller using the ATmega328P. The system accepts PET bottles, aluminum cans, and glass bottles, and rewards users with a coupon in exchange.

🔧 **Features**
* Material detection using: Infrared (IR) sensors, Capacitive sensors, Inductive proximity sensors
* Automated classification of inserted items
* Reward based system
* Custom PCB design and implementation

🛠️ **Development Tools**
* Proteus – circuit simulation and prototyping
* Eagle CAD – PCB design and 3D visualization
* Arduino (C/C++) – embedded system programming

📁 **Repository Contents**
* Source code for the Arduino-based controller
